* Python 3+

* Simple JWT
* Sqlite

### Instalar

Crear un entorno virtual con virtualenv e instalar paquetes

```
virtualenv venv
source venv/bin/activate

pip install -r requirements.txt
```

### Ejecutar la app

```
python manage.py migrate
python manage.py runserver
```
